# MusicApp
Музыкальный плеер, использующий Itunes API для поиска музыки.

Стэк:
* Swift, UIKit, AutoLayout
* URLSession
* GCD
* CocoaPods
* SDWebImage
* Realm
* RxSwift, RxCococa
